# 2D: Kaskat Asansör

### Rigging

Asansör rigging, tasarım sürecinin çoğunu kaplar, çünkü asansörün en önemli parçasıdır. Rigging tarzı ve organizasyon, motor montajını ve kelepçe plakasının monte edildiği çapraz üyenin konumunu sürer, bu yüzden kablo yollarını düzgün planlamak önemlidir.
