# Gerekli Kurs Araçları

## FRCDesignApp Kurulumu

FRCDesignApp, FRC için tasarım yaparken yararlı olan büyük bir parça kitaplığını kullanmanıza olanak tanıyan Onshape için bir eklentidir.

!!! Tip
    FRCDesignApp, belgelerinize bileşen eklemenize yardımcı olan eklenti/uygulamadır, FRCDesignLib ise bileşenlerin gerçek koleksiyonudur. FRCDesignApp gelecekte daha fazla işlevi içerecek şekilde yükseltilebilir.

Kurmak için, Onshape Appstore'daki [FRCDesignApp sayfasına](https://appstore.onshape.com/apps/Manufacturers%20Models/2ZT7X5D646R3LM3ZND7LGBTYRVM4SVH6CDDGM6I=/description "FRCDesignApp Store Page"){:target="_blank"} gidin.

"Subscribe" (Abone Ol), ardından "Get for Free" (Ücretsiz Al) butonuna basın. Bu, FRCDesignApp'ı otomatik olarak Onshape hesabınıza ekleyecektir.

!!! Note "Notlar"
    - Abone olduktan sonra, FRCDesignLib inserter'ı zaten açık olan belgelerde görünmeye başlamadan önce yenilemeniz gerekecektir. Bu bir seferlik bir şeydir.
    - İlk açtığınızda site verilerinizi (önbellek/çerezler) temizlemeniz ve uygulamayı yetkilendirmeniz gerekebilir
